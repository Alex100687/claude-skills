# Блок 2 — tmux

## Зачем tmux
WiFi моргнул, ноутбук заснул, Termius закрылся — без tmux всё что работало на сервере умирает. tmux создаёт сессию на сервере, которая живёт независимо от подключения.

## Золотое правило
Подключился к серверу → сразу `tmux` (или `tmux attach` если сессия уже была).

## Основные команды
```bash
tmux                    # новая сессия
tmux attach             # вернуться к сессии
tmux ls                 # список сессий
tmux new -s work        # именованная сессия
tmux attach -t work     # подключиться к именованной
tmux kill-session -t work  # убить сессию
```

## Горячие клавиши (Ctrl+B, затем клавиша)
| Комбинация | Действие |
|---|---|
| Ctrl+B, c | Новое окно |
| Ctrl+B, n | Следующее окно |
| Ctrl+B, p | Предыдущее окно |
| Ctrl+B, % | Разделить вертикально |
| Ctrl+B, " | Разделить горизонтально |
| Ctrl+B, стрелки | Перемещение между панелями |
| Ctrl+B, d | Отключиться (detach) — сессия живёт |
| Ctrl+B, [ | Режим прокрутки (выход — q) |

## Типичный рабочий день
```bash
# Утро
tmux
cd /opt/workspace && claude    # Окно 1: Claude Code
# Ctrl+B, c
cd /opt/workspace && codex     # Окно 2: Codex
# Ctrl+B, n/p — переключение
# Ctrl+B, d — отсоединяемся

# Вечер с телефона
tmux attach                    # всё на месте!
```
