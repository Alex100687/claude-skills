# Блок 3 — Claude Code

## Установка
```bash
npm install -g @anthropic-ai/claude-code
```

## Первый запуск
```bash
cd /opt/workspace
claude
```

## OAuth-авторизация
1. Claude Code покажет URL (начинается с `https://console.anthropic.com/...`)
2. Скопировать URL и открыть в браузере **на ноутбуке** (не на сервере!)
3. Войти в аккаунт Anthropic
4. Подтвердить авторизацию
5. Скопировать код и вставить обратно в терминал сервера

## Настройка языка
```bash
claude config set language ru
```

## Основные команды внутри Claude Code
| Команда | Описание |
|---|---|
| /help | Список всех команд |
| /model | Показать/переключить модель |
| /status | Статус подключения и подписки |
| /compact | Сжать контекст (экономит токены) |
| /clear | Очистить текущий диалог |
| /permissions | Управление разрешениями |

## Тестовый промпт
```
Привет! Создай скрипт на Python, который показывает погоду в Москве через wttr.in API
```

## Важно
Claude Code работает в контексте текущей директории. Всегда `cd` в нужную папку перед запуском.
