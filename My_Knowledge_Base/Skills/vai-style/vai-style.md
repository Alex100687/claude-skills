# VAI Style — Стиль @VAI_ART

Переформатирует список AI-новостей в авторский стиль Telegram-канала @VAI_ART.

Автор — CG-специалист с 15 годами опыта. Пишет для своей аудитории напрямую, без пафоса. Тон: "свой парень, который в теме". Аудитория знает терминологию — разжёвывать не нужно.

---

## Шаг 1: Вступление

Выбери одну из фраз-открывашек в зависимости от контекста недели. Это реальные фразы автора, они должны ощущаться живыми, а не шаблонными.

**Если неделя насыщена новостями:**
> Рубрика самых интересных новостей в ИИ за неделю. Все по плану. Погнали!

**Если неделя нейтральная:**
> Не меняя традиции. Пробежимся по новостям за неделю.

**Если новостей особенно много или неделя взрывная:**
> Ох. Происходит очень много всего и везде. Быстро пробежимся по самому интересному.

**Если неделя спокойная (мало новостей):**
> Немного ленивая неделя. На следующей побольше будет всего.

Это шаблоны, а не жёсткие правила. Можно чуть адаптировать под реальный контекст — но сохраняй тон: коротко, живо, без канцелярщины.

---

## Шаг 2: Список новостей

Каждый пункт — одна новость. Формат:

```
🔹 [Название модели или продукта] — [что вышло и чем примечательно, 1–2 предложения]
```

**Между пунктами — пустая строка.**

### Что важно в каждом пункте

- Начинай сразу с сути — что именно вышло, что изменилось
- Если есть конкретная цифра (цена, длина видео, скорость) — добавь
- Можно добавить короткую личную ноту: "выглядит очень круто", "интересная штука", "стоит попробовать" — но не злоупотребляй, это должно быть органично
- Нет подзаголовков, нет bold-текста, нет вложенных списков

### Примеры пунктов

```
🔹 Kling 3.0 Motion Control — Kuaishou добавили перенос движений из референс-видео на персонажа. До 30 секунд, 1080p, физика работает корректно.

🔹 Nano Banana 2 — Google обновили модель генерации изображений. Чище, точнее с текстом, держит консистентность до 5 персонажей. Стоит вдвое дешевле предыдущей.

🔹 GPT-5.3 Instant — OpenAI выкатили обновление. Меньше отказов, лучше синтезирует из веб-поиска. Уже в ChatGPT и Microsoft 365 Copilot.
```

## Шаг 3: Закрытие

В конце всегда:

```
@VAI_ART
#VAI_News
```

Если есть что добавить (аудиоверсия, ссылка на следующий выпуск, уточнение) — добавь P.S. перед подписью:

```
P.S. Аудиоверсия в комментариях.

@VAI_ART
#VAI_News
```

## Итоговый шаблон

```
[Вступление — 1–2 предложения]

🔹 [Название] — [описание]

🔹 [Название] — [описание]

🔹 [Название] — [описание]

...

[P.S. — если нужен]

@VAI_ART
#VAI_News
```

---

## Что НЕ делать

- **Не использовать шапку** типа "ИИ-дайджест | дата" — это не стиль автора
- **Не использовать bold** (**жирный текст**) внутри пунктов
- **Не делать подзаголовки** между пунктами
- **Не писать nested bullets** (список внутри пункта) — только 🔹 на верхнем уровне
- **Не использовать маркетинговые фразы**: "революционный", "инновационный", "прорывной"
- **Не перегружать эмодзи** — только 🔹 в пунктах списка

---

## Лимит символов

Telegram принимает до 4096 символов. Если пост превышает лимит — сокращай описания пунктов, начиная с менее важных новостей. Лучше 6 хороших пунктов, чем 10 обрезанных.
