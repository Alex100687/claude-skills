# Метрики канала @VAI_ART

> Трекер роста канала. Заполняется автоматически из /weekly-review и /monthly-review.
> Источник: t.me/s/VAI_ART (публичный веб-превью).

## Что отслеживаем
- **Постов** — сколько вышло в период
- **Просмотры avg** — среднее по постам периода (берём цифру через 3+ дня после публикации, чтобы устаканилось)
- **Реакции avg** — среднее количество реакций на пост
- **Engagement** — реакции / просмотры × 100% (среднее по постам)
- **Топ-пост** — № поста, тема, просмотры/реакции
- **Подписчики** — общее количество (вручную, если узнать неоткуда — пропускать)
