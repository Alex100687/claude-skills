# Создание и перенос префабов

> Из Todoist (Работа → Информация) — 2026-04-12

## Перенос в префаб

1. Перекидываем модель (без изменений относительно изначальной)
2. Открываем префаб (двойной клик)
3. Перекидываем меш с нового объекта в Mesh

## Создание префаба

1. Перекидываем модель (объект, не меш) в сцену
2. Из Hierarchy перекидываем в папку Prefabs
