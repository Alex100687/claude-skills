Рубрика самых интересных новостей в ИИ за неделю. Все по плану. Погнали!

🔹 Photoshop Beta — Rotate Object
В бете появился поворот объектов как 3D. AI достраивает скрытые стороны, Harmonize подстраивает освещение под сцену. Работает прямо в слоях.

🔹 <a href="https://www.hitem3d.ai/">Hitem 3D V2</a> — генерация до 2M полигонов
Обновили генератор 3D-моделей. Новая фича Photo to STL — загружаешь фото, получаешь модель для 3D-печати.
<blockquote>— 2 миллиона полигонов — детализация на уровне скульпта
— Photo to STL — из фотографии сразу в принтер без ручной обработки</blockquote>

🔹 <a href="https://3d.hunyuanglobal.com/studio">HY 3D Studio</a> — Tencent запустили глобальную 3D-платформу
Полный пайплайн: фото → 3D → топология → UV → текстуры → риг → анимация. 20 бесплатных генераций в день.
<blockquote>— из одной фотки до готовой анимированной модели
— автоматический ретопо и UV — можно сразу в движок</blockquote>

🔹 Sora References — консистентные персонажи в видео
Sora добавили References для контроля внешнего вида между кадрами. Похоже на то, что уже есть в Kling.

🔹 Grok 1.5 — до 7 референсов в генерации
В Grok теперь можно скармливать до 7 картинок-референсов одновременно. Работает и для фото, и для видео.
<blockquote>— задаёшь стиль, персонажа и окружение одновременно через несколько референсов
— работает в связке фото + видеогенерация</blockquote>

🔹 Krea Edit — камера, релайт, ререндер
Новая версия Krea Edit. Управление камерой как в лорах для Wan, плюс релайтинг и ререндер сцены.

🔹 <a href="https://cascadeur.com/">Cascadeur</a> — AI-интерполяция поз
Задаёшь две позы — AI генерирует промежуточную анимацию. Всё локально, без подписки. Плюс OpenPose-скелет для ControlNet.
<blockquote>— две ключевые позы → готовая анимация без ручного сплайна
— рендерит OpenPose RGB-скелет — сразу в ControlNet для генерации</blockquote>

🔹 Runway Characters — real-time AI-аватары через API
Runway запустили интерактивные аватары. Встраиваются в сайты и приложения, реагируют на камеру в реальном времени. $10 за 1000 кредитов.

🔹 ComfyUI App Builder — воркфлоу в приложение
Берёшь любой воркфлоу, выбираешь входные параметры — они становятся слайдерами. На диске тот же JSON, но с UI-обёрткой.
