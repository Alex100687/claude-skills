tg-ai-digest:
Создаёт еженедельный дайджест продуктовых AI-новостей из Telegram-каналов — только выход новых моделей и обновления существующих. Использовать когда пользователь просит сделать дайджест, выпуск или подборку AI-новостей за неделю, собрать посты из телеграм-каналов. Форматирует готовый пост в пределах 4096 символов.

vai-style:
Форматирует AI-новости и дайджесты в личном стиле Telegram-канала @VAI_ART. Использовать когда пользователь просит оформить пост "в моём стиле", "под VAI_ART", "как в канале", или когда нужно применить авторский стиль к готовому списку новостей. Автоматически применять после tg-ai-digest, если пользователь хочет стиль своего канала. Также использовать при любых запросах на создание поста для канала VAI_ART.
